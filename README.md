
**RAG-бот по стандарту C 11**: отвечает на вопросы по языку C строго по стандарту 2011 года, на основе RAG подхода."

***

## Запус проекта

### 1. Клонируйте репозиторий

```bash
git clone git@github.com:Mezhbert/qa_rag_bot.git
cd qa_rag_bot
```


### 2. Настройте переменные окружения

Создайте `.env` по примеру:

```
TELEGRAM_BOT_TOKEN=ваш_токен_тут
```

### 3. Убедитесь, что установлены Docker и Docker Compose

### 4. Соберите и запустите сервисы

```bash
docker compose up --build
```

### 5. Откройте Telegram и найдите вашего бота

Команда `/start` начнёт работу с ботом.
